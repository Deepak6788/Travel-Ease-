About TravelEase

TravelEase is a travel booking web application developed as a demonstration project using Flask, HTML, CSS, JavaScript and SQLite. It provides a simple platform where users can explore destinations, create an account, log in securely and manage travel bookings.

The project demonstrates modern web development concepts including user authentication, database integration, responsive design and dynamic content rendering using Flask templates.
